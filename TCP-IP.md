## TCP/IP

|        |                                       |
| ------ | ------------------------------------- |
| 应用层 | 报文(message)                         |
| 传输层 | 报文段(segment)                       |
| 网络层 | 分组/数据包(packet)、数据报(datagram) |
| 链路层 | 帧(frame)                             |
| 物理层 | bit流(bit)                            |
